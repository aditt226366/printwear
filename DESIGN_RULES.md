# Design Rules

Never use:
- KPI cards in rows
- Bootstrap appearance
- AdminLTE appearance
- Generic CRM templates
- Giant empty containers
- Repeated card grids
- Dashboard clones

Must feel like:
- Linear
- Attio
- Stripe
- Mercury
- Clay

Every page must have:
1 primary action
1 primary insight
1 memorable design element

Focus:
- hierarchy
- typography
- spacing
- composition
- visual identity
